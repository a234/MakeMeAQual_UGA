Make Me a Qual Inspired by / adapted from Jonathan Love's tool of the same name . Instructions:
  • There is an underlying collection of questions from a large number of quals, the options here essentially filter these down and take a random selection.
  • Select exam type (Algebra, Analysis, etc). The later selections will be populated with the metadata from the available questions.
  • Use the "Topics" and "Years" sections to add or remove questions, while ensuring the "Number of Matching Questions" is strictly positive.
  • Choose the number of questions to appear on the exam, then click "Make Me a Qual" to open a new HTML window with the exam questions.
  • (Note that you can instead generate a PDF to download, but this takes slightly longer.)
